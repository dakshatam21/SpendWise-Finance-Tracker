# 💸 SpendWise – Personal Finance Tracker

SpendWise is a modern and responsive personal finance management web application that helps users track income, expenses, budgets, savings goals, and monthly spending patterns.

Built using pure HTML, CSS, and JavaScript without any external frameworks.

---

## 🚀 Features

### 📊 Dashboard

* Financial overview with income, expenses and balance
* Monthly statistics cards
* Spending breakdown visualization

### 💸 Transaction Management

* Add income and expense records
* Edit existing transactions
* Delete transactions
* Category-based organization

### 🎯 Savings Goals

* Create savings goals
* Track progress visually
* Add funds toward goals
* Goal completion tracking

### 🗂 Budget Management

* Set monthly spending limits
* Monitor category-wise spending
* Budget utilization indicators

### 📈 Spending Insights

* Category-wise expense analysis
* Smart spending observations
* Monthly financial trends

### 🔁 Recurring Payments

* Track subscriptions and fixed monthly expenses
* Manage recurring transactions

### ⚙ Settings

* Light and Dark Theme
* Currency selection
* Profile customization
* CSV Export functionality

### 📱 Responsive Design

* Desktop sidebar navigation
* Mobile bottom navigation
* Fully responsive layout

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
* Canvas API

---

## 📂 Data Storage

All user data is stored locally using browser Local Storage. No backend or database is required.
